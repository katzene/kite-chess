# Smyslov Chess Engine
A UCI chess engine which analyzes and attempts to output the best moves.

## Build Instructions

Smyslov Chess requires a C++14 compiler and CMake. Run these commands to build.

```
mkdir build
cd build
cmake ..
make
```
