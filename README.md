# Kite Chess Engine

## About
Kite is a work in progress UCI chess engine which analyzes and outputs the best moves.

## Build Instructions

Kite requires a C++ 17 compiler and CMake 3.22. Run these commands to build:

```
mkdir build
cd build
cmake ..
make
```
